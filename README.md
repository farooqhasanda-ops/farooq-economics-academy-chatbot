# Farooq Economics Academy AI Assistant

A simple Gradio chatbot for Farooq Economics Academy.

## Run locally

```bash
pip install -r requirements.txt
python app.py
```

Open: `http://localhost:7860`

## Deploy on Hugging Face Spaces

1. Create a new Space on Hugging Face.
2. Select **Gradio** as the SDK.
3. Upload `app.py`, `requirements.txt`, and `README.md`.
4. Commit the files.
5. Your chatbot will be live automatically.

## Deploy on Render

Start command:

```bash
python app.py
```

Port: `7860`
